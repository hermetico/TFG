=================================================
Bienvenido a la documentacion del Annotation tool
=================================================

Annotation tool es una aplicación que esta pensada para etiquetar imagenes de forma rapida. La información
de las imagenes y sus etiquetas estan guardadas en una base de datos  `SQLite <https://www.sqlite.org>`_ .
La aplicación incluye scripts para exportar las etiquetas y los datasets. Además, esta diseñada para 
trabajar con imagenes de la camara `Narrative Clip <http://getnarrative.com>`_ con lo cual incluye una serie de
scripts para ser usados con `Matlab <http://es.mathworks.com/products/matlab/>`_  que son ejecutados desde 
la misma consola de linux en la que esta el servidor funcionando.
